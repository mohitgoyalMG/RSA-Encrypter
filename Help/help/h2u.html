First frame of the application shows Four options:

  1. Generate
  2. Encrypt
  3. Decrypt
  4. Help


    5. Choose the appropiate option from the frame

    1. Generate option - Click on 'Generate' button on the Generate frame to generate the keys. Copy all the keys and keep them in a safe place.
    2. Encrypt option -
      1. Enter the text that you want to encrypt.
      2. Enter 'Part 1' and 'Part 2' of public key that you generated previously.
      3. Click on 'Encrypt' button to get the encrypted text.
      4. Copy the encrypted text and keep it in the safe place.
    3. Decrypt option -
      1. Enter the encrypted text that you want to decrypt.
      2. Enter 'Part 1' and 'Part 2' of private key that you generated previously.
      3. Click on 'Decrypt' button to get the original text.
    4. Help option - Goto this option if you need some help regarding the application.
